# RF-14.2: Comparação de Fotos de Progresso — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Adicionar comparação lado a lado de duas fotos de progresso em `PhotosView.vue` — seleção por clique no grid, barra contextual e dialog de comparação.

**Architecture:** Modificação de um único arquivo Vue (`PhotosView.vue`). Novos refs `selectedPhotos` e `showCompare`, computed `comparePhotos`, funções `toggleSelect` e `isSelected`. Alterações no template: cada card do grid ganha click handler + anel de seleção + checkmark; nova barra de ações contextual; novo Dialog PrimeVue com layout duas colunas.

**Tech Stack:** Vue 3 + TypeScript, PrimeVue (Dialog, Button), Tailwind CSS, Vite dev server (`cd frontend && npm run dev`)

## Global Constraints

- PrimeVue components importados individualmente: `import Dialog from 'primevue/dialog'`
- `Photo` type: `{ id: string; studentId: string; uploadedBy: string; takenAt: string; notes: string | null; fileUrl: string; createdAt: string }`
- Dialog responsivo: `style="width: min(860px, 95vw)"`
- Máximo 2 fotos selecionadas — clicar em 3ª não faz nada
- Foto mais antiga à esquerda ("Antes"), mais recente à direita ("Depois") — ordenado por `takenAt`
- Fechar dialog via X ou "Fechar" mantém a seleção; "Limpar seleção" deseleciona tudo e fecha
- Botão delete fica oculto (`v-show`) quando a foto está selecionada
- Commit em português com prefixo `feat:`
- Sem novos arquivos, sem novo store, sem alterações no backend

- [ ] **Step 2: Adicionar estado de seleção e comparação**

Após `const fileInput = ref<HTMLInputElement | null>(null)` (linha 39), adicionar:

```ts
const selectedPhotos = ref<Photo[]>([])
const showCompare = ref(false)

function isSelected(photo: Photo): boolean {
  return selectedPhotos.value.some(p => p.id === photo.id)
}

function toggleSelect(photo: Photo) {
  const idx = selectedPhotos.value.findIndex(p => p.id === photo.id)
  if (idx >= 0) {
    selectedPhotos.value.splice(idx, 1)
  } else if (selectedPhotos.value.length < 2) {
    selectedPhotos.value.push(photo)
  }
}

const comparePhotos = computed(() =>
  [...selectedPhotos.value].sort(
    (a, b) => new Date(a.takenAt).getTime() - new Date(b.takenAt).getTime()
  )
)

function clearSelection() {
  selectedPhotos.value = []
  showCompare.value = false
}
```

  >

  </div>
